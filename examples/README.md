# AstraOS Examples

示例应用程序展示 AstraOS 的各种功能。

## 运行示例

### Runtime 示例

```bash
cd os/examples/runtime_example
cargo run
```

这个示例展示了：
- 任务调度和执行
- 资源管理
- Actor 系统
- 健康监控

### Kernel 示例

```bash
cd os/examples/kernel_example
cargo run
```

这个示例展示了：
- 节点创建和管理
- 服务发现
- 配置更新
- 生命周期控制

## 自定义示例

可以基于这些示例创建自己的应用程序：

1. 复制示例目录
2. 修改代码实现自己的功能
3. 添加依赖到 Cargo.toml
4. 运行和测试
